Cookie Policy

Privana keeps browser storage minimal. We use only what is needed for login, security, theme preference, and basic app functionality.

1. Essential cookies and storage

Privana uses essential session and security storage to keep you logged in, protect forms, remember theme preference, and support the anonymous account flow.

2. What may be stored

Name / Type Purpose Category Duration
Session cookie Keeps your anonymous account session active. Essential Session / configured lifetime
CSRF token Protects forms and browser actions from cross-site request forgery. Essential Session
theme Remembers light or dark mode preference. Preference Until cleared
PWA/browser storage Supports install prompts and app-like behavior where available. Functional Browser controlled

3. Analytics

The authenticated Privana app does not require third-party analytics. If privacy-friendly analytics are added to public pages later, this policy should be updated and controls should be provided where required.

4. How to control storage

You can clear cookies and local storage from your browser settings. Clearing essential storage may log you out or reset interface preferences.

For cookie or storage questions, contact GoAnon through GoAnon.pro.
